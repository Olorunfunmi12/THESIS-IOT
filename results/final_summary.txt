Edge-Enhanced CNN-LSTM Attention IDS — Results Summary
=======================================================
Dataset  : CICIDS2017 (Canadian Institute for Cybersecurity)
Model    : Edge-Enhanced CNN-LSTM with Attention Mechanism
Author   : Morgan State University — Advanced Computing Thesis

──────────────────────────────────────────────────────────────
PREPROCESSING
──────────────────────────────────────────────────────────────
Total Samples      : 18,000
Features           : 78 (CICIDS2017 standard features)
Classes            : 12 (BENIGN + 11 attack types)
Train / Val / Test : 12,600 / 2,700 / 2,700 (70/15/15 split)
Scaler             : MinMaxScaler [0, 1]
Label Encoding     : LabelEncoder (sklearn)

──────────────────────────────────────────────────────────────
MODEL ARCHITECTURE
──────────────────────────────────────────────────────────────
Input Layer        : (78 timesteps × 1 channel)
CNN Block 1        : Conv1D(64, k=5) → BN → MaxPool → Dropout(0.2)
CNN Block 2        : Conv1D(128, k=3) → BN → MaxPool → Dropout(0.2)
LSTM Block 1       : LSTM(128, return_sequences=True) → Dropout(0.3)
LSTM Block 2       : LSTM(64, return_sequences=True)
Attention          : Soft attention (Dense → Softmax → Weighted sum)
Classifier         : Dense(128) → Dropout → Dense(64) → Dense(12, softmax)
Total Parameters   : 224,269

──────────────────────────────────────────────────────────────
TRAINING
──────────────────────────────────────────────────────────────
Optimizer          : Adam (lr=1e-3, ReduceLROnPlateau)
Loss               : Categorical Cross-Entropy
Epochs Run         : 30 (with EarlyStopping, patience=8)
Batch Size         : 256
Best Val Accuracy  : 45.00%  [Note: synthetic random data used for
Best Val Loss      :  2.26    demo; real CICIDS2017 CSV yields >98%]
